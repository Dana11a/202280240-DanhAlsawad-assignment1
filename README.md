# Assignment 1 – Personal Portfolio

##  Project Description
This project is a responsive personal portfolio website built using HTML, CSS, and JavaScript.  
The website includes an About section, Projects section with placeholder images, and a Contact form.

The design supports both Dark and Light mode and adapts to desktop, tablet, and mobile screens.

---

## Features
- Responsive design (mobile-friendly)
- Dark / Light mode toggle
- Styled content sections with modern layout
- Two project sections with images
- Contact form (front-end only)
- Clean folder structure

---

## Technologies Used
- HTML5
- CSS3 (Flexbox, Gradients, Responsive Design)
- JavaScript (DOM Manipulation, Theme Toggle)

## How to Run Locally

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Resize the browser window to test responsiveness.
4. Use the toggle button to switch between dark and light modes.
